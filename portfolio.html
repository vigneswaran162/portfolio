Description: Virtual Event Management System Our Virtual Event Management System revolutionizes college event planning. Authorized staff can create, manage, and promote events, while students receive automated email confirmations with QR code tickets. Role-based access ensures secure management, and real-time updates keep everyone informed. The system also allows uploading old events to track progress and analyze trends. It streamlines event organization, reduces administrative tasks, and enhances the overall experience for both staff and students.

Technologies used:

  • Frontend: Angular + Bootstrap → Ideal for building a responsive and dynamic user interface.
  • Backend: Express.js + Node.js → Manages server-side operations, API requests, and handles business logic.
  • Database: MongoDB → Suitable for storing event details, user information, and logs due to its flexibility and scalability.
